# ProvisionEdge

ProvisionEdge is a sovereign infrastructure consultancy. The site is built with Astro and Tailwind CSS, adapted from the `alfadev-astro-starter` theme.

## Mission

ProvisionEdge exists to build local-first, decentralized systems that ensure operational continuity, data integrity, and digital autonomy.

## About this site

ProvisionEdge.com is a static site that communicates our mission, services, and principles for sovereign infrastructure.

For technical build and deployment instructions, see [`DEPLOYMENT.md`](DEPLOYMENT.md).
